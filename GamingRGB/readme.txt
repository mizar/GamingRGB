# 使い方
1. 同梱の GamingRGB.shader を Assets のお好きな場所にインポート
2. 適用したいマテリアルのシェーダーとして Custom/GamingRGB を選択
3. 光ります

# 設定
- Decolor で脱色の有無を指定します。脱色したほうが概ねキレイになります。
- Red, Green, Blue の値で各色の強さを指定します。
- Cycle は色の変わる周期です。
- Scale は色の範囲の大きさです。
- RGB_Intensity, Texture_Intensity は光とテクスチャの濃さです。
  テクスチャなし、あるいは白いテクスチャの場合、Texture_Intensity を小さくすることをお勧めします。
- Freq_... は各色の周期を変更するものです。変に弄ると汚い色になります。
